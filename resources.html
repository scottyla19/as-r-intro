4.1 R Packages

We need to install a package before so that we can complete a minilab. We can think of a package like an app on our phone. When you first turn on a new phone you have the basic functionality that all other phones have. If you need more functionality, you install new apps. Packages are to R as apps are to your phoneâ€™s operating system. The swirl package is a great way to learn how to use the R programming language. Install the package, run it, and learn R in the R console! Before we start we need to download a swirl course that we can use to test our understanding so far. [Download this file]

Type the following code in the console. Notice once you start typing RStudio provides hints. You can press tab to select the function you want. We need to first download and install the package with install.packages() function.

install.packages('swirl')
## Error in install.packages : Updating loaded packages

The code above will start a process that will download and install the necessary parts so that we can use the swirl package.

Once the package is installed we need to tell R to use it with the library() function.

library(swirlify)

Now we can start the swirl app with the swirl() function.

swirl()
